# LeetSync

Turn your LeetCode solutions into an organized GitHub portfolio.

## Features

- Automatic GitHub synchronization
- Custom repository selection
- Organized problem folders
- Smart commit messages
- Submission history
- Secure GitHub authentication

## How It Works

1. Install LeetSync
2. Connect your GitHub account
3. Select or create a repository
4. Solve a LeetCode problem
5. LeetSync automatically uploads the accepted solution

## Privacy

LeetSync does not collect or sell personal information.
Authentication credentials are securely stored and used only
for GitHub synchronization.
